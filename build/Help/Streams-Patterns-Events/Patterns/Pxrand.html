Pxrand



like Prand, returns one item from the list at random for each repeat, but Pxrand never

repeats the same element twice in a row. 


(

var a, b;

a = Pxrand.new(#[1, 2, 3], 10); // return 10 items

b = a.asStream;

11.do({ b.next.postln; });

)
