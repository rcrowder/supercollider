Color an object which stores color values


Inherits from: Object


An object which stores red, green, blue, and alpha values in instance variables, and  provides numerous class and instance methods for creating and manipulating these values.


See also: Object 


Creation / Class Methods


*new (red, green, blue, alpha)


red - An instance of Float between 0 and 1. Default value is 0.0

green -  An instance of Float between 0 and 1. Default value is 0.0

blue -  An instance of Float between 0 and 1 . Default value is 0.0

alpha -  An instance of Float between 0 and 1 . Default value is 0.0


*new255 (red, green, blue, alpha)

RGB values from 0-255.  Takes 8-bit values as arguments.

red - An instance of Integer between 0 and 255. Default value is 0.0

green -  An integer between 0 and 255. Default value is 0.0

blue -  An integer between 0 and 255 . Default value is 0.0

alpha -  An integer between 0 and 255 . Default value is 255


*fromArray (array)

array - An instance of Array. 1-4 values between 0 and 1. See new. Same as *new(*array)


*black - nothing is darker.

*white - nothing is lighter.

*clear - nothing is more clear.

*gray (gray, alpha)

*grey (grey, alpha)

Brittish or American shades of grey.

gray/grey  -  shades between 0 (black) and 1 (white) as Float. Default is 0.5.

alpha - Transparency of grey between 0 and 1 as Float. Default is 1.

*red (val, alpha)

*green (val, alpha)

*blue (val, alpha)

*cyan (val, alpha)

*magenta (val, alpha)

*yellow (val, alpha)


val - Default is 1. An instance of Float between 0 and 1, where 1 is the bright, fully saturated color, and 0 is black.

alpha - Transparency between 0 and 1 as Float. Default is 1.


*rand (lo, hi)

A random RGB Color where the colors are randomly chosen between lo and hi. See new.

lo - An instance of Float between 0 and 1. Default value is 0.3.

hi - An instance of Float between 0 and 1. Default value is 0.9.


*hsv (hue, sat, val, alpha)

Maps hsv and alpha values to rgb between 0 and 1.

hue - Default value is nil. Float between 0 and 0.999.  Default value is nil.

sat - Explanation of sat. Float between 0 and 1. Default value is nil.

val - Explanation of val. Float between 0 and 1.  Default value is nil.

alpha - Explanation of alpha. Default value is 1. Float between 0 and 1.

// returns true:

blend (that, blend)

Interpolates the red, green, blue, and alpha values of two Colors,  linearly weighting between the first and the second.

that - A second Color.

blend - 0 menas only this, while 1 means only that. default is  0.5 

vary (val, lo, hi, alphaVal)

Returns a random variation of a Color.

val - Varies the RGB values +- this amount. Default value is 0.1.

lo - Low-clips the RGB variation here . Default value is 0.3..

hi - High-clips the RGB variation here . Default value is 0.9.

alphaVal - Varies the alpha value +- tis amount, and clips it between 0 and 1.
