TabletView a cross-platform redirector for a GUI widget


Inherits from: Object : ViewRedirect


This class will return the gui kit specific version of a TabletView.


See also: GUI, GUI-Classes, GUI-Overview


Platform Specific Examples and Documentation:


SwingGUI: JSCTabletView


CocoaGUI: SCTabletView


Some of the links above might not work, if the help files for the listed kit are not included in your setup. If you do not see your GUI kit listed in the help redirect stubs, then execute the following code to regenerate the GUI stubs. Your kit must be registered with GUI in order to be listed in the redirect stubs. See GUI on how to do this.

(

Helper.updateRedirectStubs; // this may take a moment to complete

Document.current.close; // will close this document when the update is complete

)
