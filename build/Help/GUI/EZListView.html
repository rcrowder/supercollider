EZListView a wrapper class for a label plus a listView with per item actions


Inherits from: Object


EZListView is wrapper class which creates an (optional)  label and a listView. It includes  per item actions as well as a global action which are both evaluated upon selection of an item. Convenience methods for inserting and deleting menu items are also included . 


See also: SCListView  


Some Important Issues Regarding EZListView


The convenience methods for EZListView require that the items array is an array of associations  of  labels and  functions, not like in SCListView, where items is simply an array of strings. If label is nil, then no staticText is created. If you set label later, a staticText will be created.


Creation / Class Methods


*new (parentView, bounds, label, items, globalAction, initVal, initAction, labelWidth)

parentView - The parent view or window. Default value is a FlowView window.

bounds -  An instance of Rect or Point. Default value is 160@16.  The Menu width is bounds.x- labelWidth

label - the label. Default value is nil. If nil, then no SCStaticText is created.  If you set label later, the labelView will be created.

items - Default value is nil. An Array of Associations ['label' -> {arg menuObj; value}, ].

globalAction - a Global function to be performed in addition to the item functions {arg menuObj; value}.

initVal - Initial value of the Menu, i.e. the index selected. Default value is 0. 

initAction - An instance of Boolean. Performs the action at initVal on creation of the menu, plus the globalAction. Default value is false.

labelWidth - Default value is 80. Not used if labelPosition is top.

labelHeight - Default value is 20. Not used if labelPosition is  left.

labelPosition- \top of \left. default is left;

gap - Default value is 4. If the label is not nil, the gap between the label and the menu .

(// default with label on top

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZListView.new(w,

230@230,

"An SCListView:",

[

\item0 ->{|a| ("this is item 0").postln},

\item1 ->{|a| ("this is item 1").postln},

\item2 ->{|a| ("this is item 2").postln},

],

globalAction: {"this is  a global action".postln},

initVal: 2, 

initAction: true,

labelWidth: 120

);

)

// or a more simple syntax:

(

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZListView.new(w,200@230, "List:");

g.addItem(\item0, {"this is item 0". postln});

g.addItem(\item1, {"this is item 1". postln});

g.addItem(\item2, {"this is item 2". postln});

)

// label on the left:

(

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZListView.new(w,200@230, "A List Label:", labelPosition:\left);

g.addItem(\item0, {"this is item 0". postln});

g.addItem(\item1, {"this is item 1". postln});

g.addItem(\item2, {"this is item 2". postln});

)

// No labelView created;

(

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZListView.new(w,200@230); // no label

g.addItem(\item0, {"this is item 0". postln});

g.addItem(\item1, {"this is item 1". postln});

g.addItem(\item2, {"this is item 2". postln});

)


Building and Changing the Menu


globalAction_(function)

globalAction

a Global function to be performed in addition to the item functions: {arg menuObj; value}.


items

items_ (assocArray)

assocArray - An Array of Associations including the labels and the item functions:   ['label' -> {arg menuObj; value}, ].

addItem (name, action)

Adds an item.

name - An instance of  String or Symbol. The name of the menu item.

action - A n instanc of Function.

(

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZListView.new(w,200@200, "List:");

g.addItem(\item0, {"this is item 0". postln});

g.addItem(\item1, {"this is item 1". postln});

g.addItem(\item2, {"this is item 2". postln});

)


insertItem (index, name, action)

Inserts a menu item at positiion of index.

index - An integer. The index where to insert an item.

name - An instance of  String or Symbol. The name of the menu item.

action - A n instanc of Function.

(

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZListView.new(w,200@200, "List:");

g.addItem(\item0, {"this is item 0". postln});

g.addItem(\item1, {"this is item 1". postln});

g.addItem(\item2, {"this is item 2". postln});

g.addItem(\item4, {"this is item 4". postln});

)

g.insertItem(3, \item3, {"this is item 3". postln});


removeItemAt (index)

Removes a menu item at positiion of index.

index - An integer. The index where to remove an item.

(

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZListView.new(w,200@200, "List:");

g.addItem(\item0, {"this is item 0". postln});

g.addItem(\item1, {"this is item 1". postln});

g.addItem(\item2, {"this is item 2". postln});

g.addItem(\item4, {"this is item 4". postln});

g.insertItem(3, \item3, {"this is item 3". postln});

)

g. removeItemAt(0);



remove

removes both the label and the menu from the parent view.


Accessing Values


value

value_ (val)

Gets/sets the menu to the index at value. Does not perform the action.

val - An integer.


valueAction_ (val)

Sets the value and performs the action at the index value and the global action.

val - An integer.

doAction

Performs the action at the current index and the global action.

Accessing GUI options


label

label_(string)

Sets/gets it the label. Will add the label view if none was initially created.

string - An Instance of String. 


visible

visible_(bool)

Sets/gets it the component  views are visible.

bool - An Instance of Boolean. Default is true;

enabled

enabled_(bool)

Sets/gets it the menu  is enabled.

bool - An Instance of Boolean. Default is true;

visible

visible_(bool)

Sets/gets it the component  views are visible.

bool - An Instance of Boolean. Default is true;

bounds

bounds_(rect)

Sets/gets the bounds of view, and adjusts the components.

rect - An Instance of Rect or Point;


labelPosition

labelPosition_(pos)

Sets the label position.  The width and height of the label return to the defaults: 80@this.height or this.width@20;

pos - \left or \top


labelWidth

labelWidth_(width)

Sets/gets the width of the label to width, and adjusts the menu.bounds. (the actual width of the label will be width-gap). Does nothing if labelPosition=top;

width - A number;

labelHeight

labelHeight_(height)

Sets/gets the height of the label to height, and adjusts the menu.bounds. (the actual width of the label will be height-gap). Does nothing if labelPosition=bottom;

height - A number;


gap

gap_(val)

Sets/gets the width of the label to width-val, creating a gap. 

val - A number. default is 4.
