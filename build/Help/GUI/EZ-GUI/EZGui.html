EZGui an abstract superclass for EZ widget wrappers


Inherits from: Object


Users will not normally directly create instances of EZLists, but only use it through its subclasses. It provides the basic mechanisms for various EZ widget wrappers.


See also: EZListView, EZPopUpMenu, EZSlider, EZNumber, EZRanger
