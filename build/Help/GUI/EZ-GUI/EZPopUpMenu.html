EZPopUpMenu a wrapper class for a label plus a popUpMenu with per item actions


Inherits from: Object : EZGui : EZLists


EZPopUpMenu is wrapper class which creates an (optional)  label and a popUpMenu. It includes  per item actions as well as a global action which are both evaluated upon selection of an item. Convenience methods for inserting and deleting menu items are also included . See EZGui and EZLists for all of the options.


See also: SCPopUpMenu  


Some Important Issues Regarding EZPopUpMenu


The convenience methods for EZPopUpMenu require that the items array is an array of associations  of  labels and  functions, not like in SCPopUpMenu, where items is simply an array of strings. If label is nil, then no staticText is created. If you set label later, a staticText will be created.


Creation / Class Methods


*new (parentView, bounds, label, items, globalAction, initVal, initAction, labelWidth)

parentView - The parent view or window. Default value is a FlowView window.

bounds -  An instance of Rect or Point. Default value is 160@20. 

label - the label. Default value is nil. If nil, then no SCStaticText is created.  If you set label later, the labelView will be created.

items - Default value is nil. An Array of Associations ['label' -> {arg menuObj; value}, ].

globalAction - a Global function to be performed in addition to the item functions {arg menuObj; value}.

initVal - Initial value of the Menu, i.e. the index selected. Default value is 0. 

initAction - An instance of Boolean. Performs the action at initVal on creation of the menu, plus the globalAction. Default value is false.

labelWidth - Default value is 80. 

gap - by default, the view tries to get its parent's gap.x, otherwise it defaults to 4.  Setting it overrides these.

(

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZPopUpMenu.new(w,

230@20,

"An SCPopUpMenu:",

[

\item0 ->{|a| ("this is item 0").postln},

\item1 ->{|a| ("this is item 1").postln},

\item2 ->{|a| ("this is item 2").postln},

],

globalAction: {"this is  a global action".postln},

initVal: 2, 

initAction: true,

labelWidth: 120

);

)

// or a more simple syntax:

(

w=Window.new.front;

w.view.decorator = FlowLayout(w.view.bounds);

g = EZPopUpMenu.new(w,200@20, "Menu: ");

g.addItem(\item0, {"this is item 0". postln});

g.addItem(\item1, {"this is item 1". postln});

g.addItem(\item2, {"this is item 2". postln});

)
