RunningSum


A running sum over a user specified number of samples, useful for running RMS power windowing.Â 


Class Methods


*ar(in, numsamp=40)


in- Input signal


numsamp- How many samples to take the running sum over (initialisation time only, not modulatable)


Examples


//overloads of course- would need scaling

{RunningSum.ar(SoundIn.ar)}.play

//Running Average over x samples

(

{

var x =100;


RunningSum.ar(LFSaw.ar,x)*(x.reciprocal)

}.play

)


//RMS Power

(

{

var input, numsamp;


input= LFSaw.ar;

numsamp=30;


(RunningSum.ar(input.squared,numsamp)/numsamp).sqrt

}.play

)



//shortcut in class

{RunningSum.rms(SoundIn.ar)}.play
