SendReply  


SendReply.kr(trig, inputs, id, cmdName)


trig a non-positive to positive transition telling Poll to return a value

cmdName a string or symbol, as a message name.

inputs array of ugens

replyID integer id (similar to SendTrig)


the reply message is formatted as follows: [msg, nodeID, replyID, inputs ...]

{
