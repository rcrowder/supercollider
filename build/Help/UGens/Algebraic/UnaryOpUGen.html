UnaryOpUgen unary operator on a ugen


superclass: BasicOpUGen


see also BinaryOpUGen


*new(selector, a) return a new instance that applies the operator (selector) to the ugen (a)

normally, this is implicitly called when applying a unary operator to a UGen.

Examples for unary operations on UGens are to be found in UGen's superclass AbstractFunction.Â 

They include: neg, abs, ceil, floor, frac, squared, cubed, sqrt, exp, log, log2, log10, sin, cos, scurve, distort

// example

a = WhiteNoise.ar; // a WhiteNoise

b = a.squared; // a UnaryOpUGen.

b.operator; // squared
