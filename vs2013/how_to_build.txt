building on win64
-----------------

- first get and install:
  - visual studio 2013 community (it's free)
  - Qt 5.4 msvc2013_64, i put it in C:\Qt
  - libsndfile 64 bit, there's an installer
  - portaudio - i think you only need portaudio_x64.dll and the include folder

- edit build.bat, set the folders right.

run build.bat, this runs cmake that builds the solution.

open the solution.

In configuration manager, uncheck build for DiskIO_UGens.

hidapi
	- sclang needs it in CMakeFiles
	- Must be built linking all dependencies
  - i had to build it separately from their github

deploying
---------

deploy.bat, when set as a Post-Build-Event for ALL-BUILD, will deploy all the files to c:\supercollider.

ALL_BUILD-->preferences-->Build Events-->Post Build Event:

$(SolutionDir)..\deploy.bat $(SolutionDir) $(Configuration) $(Platform)

or manually copy into c:\supercollider the dlls.

DLLs to deploy:
Everything from QT
portaudio_x64.dll
libsndfile-1.dll
