The search box optionally takes Regular expressions

For example:

. matches any character
[0-9] matches any number
[a-zA-Z] matches any letter
+ matches the previous one or more times
* matches the previous zero or more times
^ matches beginning of string
$ matches end of string
\b matches a word boundary

Some characters need to be escaped with \, like +*?[]().

See Regular_expression on wikipedia for more info.
